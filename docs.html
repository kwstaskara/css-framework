Introduction

Pyramid is a lightweight,responsive not biased css framework built upon sass preproccessor. Moreover pyramid provides you multiple ways to design your ui based on the desired browser compatibility.That means pyramid uses modern methods(flexbox,grid) but also legacy methods(tables etc).That way you build more accessible beautiful ui.

Pyramid structure

Pyramid is using rem with an option enabled by default to provide px fallback. Root font-size is 16px. Pyramid css is being compiled using gulp.

What's inside

Pyramid provides multiple components grid system and many more,all built purely in css
