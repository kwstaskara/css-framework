Introduction

Pyramid is a lightweight,responsive not biased css framework built upon sass preproccessor. Moreover pyramid provides you multiple ways to design your ui based on the desired browser compatibility.That means pyramid uses modern methods(flexbox,grid) but also legacy methods(tables etc). We are supporters of rapid development so we created a framework that provides multiple options to build a ui and eliminate any dilemma about css features, that way you have any css feature in one tool. This is what makes pyramid a great tool for everyone.

Structure

Pyramid is using rem with an option enabled by default to provide px fallback. Root font-size is 16px. Pyramid css is being compiled using gulp. Multiple css properties and features is the concept that lives inside pyramid framework, such as flexbox tables and propably css grid.

Installation

Pyramid css can be downloaded and install it manually. Link below

Download pyramid

Then include file <link rel="stylesheet" href="pyramid.min.css">

Install with npm

npm install pyramid.ui

Install with yarn

yarn add pyramid.ui

Using cdn

    
<link rel="stylesheet" href=" https://cdn.jsdelivr.net/npm/pyramid.ui@/dist/pyramid.min.css">
