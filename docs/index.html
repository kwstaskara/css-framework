Core Features

CSS only

Mobile first css only framework. It is javascript friendly.

Not Biased

Not biased with any css feature, allowing users to choose between multiple css features depending in browser compatibility.

Multiple Components

Ready to use components such as navbars menus, cards, navbars and many more, all built with different css rules.

Responsive

Mobile first and responsive, providing great experience across different screen sizes.

Up to date

Coded in SASS preprocessor,compiled with Gulp task-runner,minified with CSSO,autoprefixed with autoprefixer,normalized with Normalizer.css

Open source

Open source tool to develop websites,web-apps and user interfaces at no cost.

Everything you need.

Pyramid UI provides everything you need to start developing web-pages and web-apps, but also it gives you the freedom you need to apply you custom color theming. Pyramid UI does not apply default colors to components as a result web-pages and apps made with the framework are not looking the same. Custom CSS may not needed.

Feature rich all in one tool

Redpyramids UI is based on multiple CSS features not just flexbox but also tables and CSS grid allowing users pick the CSS rules they desire. For this reason components are built with multiple CSS rules.

Multiple flavors!

Default with px fallbacks. Minified version included.

Autoprefixed with px fallbacks. Minified version included.

Default no px fallbacks. Minified version included. Smallest size

Autoprefixed no px fallbacks. Minified version included.

All the above but javascript frameworks friendly. New
