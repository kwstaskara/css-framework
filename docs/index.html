Css only

Pyramid is an css only framework Javascript is not required, this result in fewer requests and faster page response.

Not biased

Pyramid is not biased with any css feature, allowing users to choose between multiple css features depending in browser compatibility.

Responsive

Pyramid css is mobile friendly and responsive, providing great experience across different screen sizes.

Open source

Pyramid is open source tool to develop websites, ui, and apps at no cost.

Why pyramid

Pyramid is a new responsive css framework that aims in better browser compatibility. Pyramid is based on rem unit built upon sass preprocessor providing fallbacks for better browser support. Yes pyramid is not based on any css feature and is not obsessed with specific color values instead it provides multiple ways of doing things depending the user preference in theming and browser support. Multiple features ready such as flexbox, tables and many more!
