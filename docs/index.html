Pyramid UI

Version / mb minified

CSS framework to build beautiful user interfaces
Get started

CSS only

Pyramid is an css only framework Javascript is not required, this result in fewer requests and faster page response.

Not Biased

Pyramid is not biased with any css feature, allowing users to choose between multiple css features depending in browser compatibility.

Multiple Components

Pyramid UI comes with many ready to use components built in such as navbars menus cards and many more.

Responsive

Pyramid css is mobile friendly and responsive, providing great experience across different screen sizes.

Up to date

Coded in SASS preprocessor,compiled with Gulp task-runner,minified with CSS NANO,autoprefixed with autoprefixer,normalized with Normalizer.css

Open source

Pyramid is open source tool to develop websites, ui, and apps at no cost.

Everything you need.

Pyramid UI provides everything you need to start developing web-pages and web-apps, but also it gives you the freedom you need to apply you custom color theming. Pyramid UI does not apply default colors to components as a result web-pages and apps made with the framework are not looking the same. Custom CSS may not needed.

Feature rich all in one tool

Pyramid UI is based on multiple CSS features not just flexbox but also tables and CSS grid allowing users pick the CSS rules they desire. For this reason components are built with multiple CSS rules.
