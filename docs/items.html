Modal

A modal is a dialog box/popup window
            Pyramid allows you to customize modal whatever you want.
            Modals can contain any element.You need to implement your own js
                to open modal by adding is-active class. 
           Modals have not default size you need to use our col system.

If legacy browsers is not an issue, we recommend flexbox modal
