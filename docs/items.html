            Pyramid allows you to customize modal whatever you want.
            Modals can contain any element.You need to implement your own js
                to open modal by adding is-active class. 
           Modals have not default size you need to use our col system.

If legacy browsers is not an issue, we recommend flexbox modal

Navbars

Flexbox

Navbars are made to be modular so mobile navbars are seperated from desktop navbars.Resize window to see mobile-nav

Flexbox mobile navbars are sepereted in 3 sections to be easier to customize and they are visible only in small screens

Desktop navs become the expanded navs in smaller screens

By deafult expanded navbar is horizontal

In order to toggle mobile nav add classes listed below to desktop navs using javascript

Desktop navs become the expanded navbar in mobile

Navbar toggler classes

.toggle-nav toggle simple nav

.toggle-flex-nav toggle flexbox nav
