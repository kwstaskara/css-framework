Responsive System

With responsive system you can control html structure in differrent screen sizes

    Classes:
    col-xs for Mobile Devices
    col-sm for Tablet Devices
    col-md for Desktops
    col-lg for Widescreen Desktops
    col system starts from 1 down to 12.
    Example:
    col-xs-1 col-md-5 col-sm-12 col-lg-7

Hide elements across different devices

    Classes:
    hide-xs for Mobile Devices
    hide-sm for Tablet Devices
    hide-md for Desktops
    hide-lg for Widescreen Desktops
  For example  an element with hide-xs class will be hidden in Mobile devices
