Hide elements across different devices

    Classes:
    hide-xs for Mobile Devices
    hide-sm for Tablet Devices
    hide-md for Desktops
    hide-lg for Widescreen Desktops
  For example  an element with hide-xs class will be hidden in Mobile devices
