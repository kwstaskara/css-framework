About pyramid

Pyramid is a new responsive css framework that aims in better browser compatibility. Pyramid is based on rem unit built upon sass preprocessor providing fallbacks for better browser support. Yes pyramid is not based on any css feature and is not obsessed with specific color values instead it provides multiple ways of doing things depending the user preference in theming and browser support.
